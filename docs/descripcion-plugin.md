# Connect CRM Real State - Descripción Comercial

## Descripción Comercial Completa

**Connect CRM Real State** es un plugin profesional de WordPress diseñado para conectar y sincronizar propiedades inmobiliarias desde sistemas CRM externos directamente a tu sitio web WordPress. Es la solución perfecta para inmobiliarias, agencias de propiedades y desarrolladores que necesitan automatizar la gestión de su catálogo inmobiliario.

### ¿Qué hace el plugin?

Este plugin actúa como un puente inteligente entre tu sistema CRM inmobiliario (Inmovilla o Anaconda) y WordPress, permitiendo que todas tus propiedades se sincronicen automáticamente en tu sitio web sin necesidad de copiar y pegar información manualmente.

### ¿A qué público va dirigido?

- **Inmobiliarias y agencias de propiedades** que utilizan sistemas CRM como Inmovilla o Anaconda y necesitan mostrar sus propiedades en su sitio web WordPress.
- **Desarrolladores web** que trabajan con clientes del sector inmobiliario y necesitan una solución de integración profesional.
- **Propietarios de sitios web inmobiliarios** que buscan automatizar la actualización de su catálogo de propiedades.

### ¿Qué problema resuelve?

El sector inmobiliario requiere mantener actualizado constantemente el catálogo de propiedades en el sitio web. Hacerlo manualmente es:
- **Consume mucho tiempo**: Copiar y pegar información de cada propiedad desde el CRM al sitio web.
- **Propenso a errores**: Información desactualizada, datos incorrectos o propiedades duplicadas.
- **Ineficiente**: No escalable cuando tienes cientos o miles de propiedades.
- **Costoso**: Requiere personal dedicado solo a mantener actualizado el sitio web.

**Connect CRM Real State** resuelve todos estos problemas automatizando completamente el proceso de sincronización.

### Beneficios Clave y Funcionalidades Principales

#### 🔄 Sincronización Automática
- Conecta tu WordPress con los sistemas CRM más utilizados en España: **Inmovilla** y **Anaconda**.
- Sincronización automática de todas las propiedades desde la API del CRM.
- Actualización inteligente: detecta propiedades nuevas, modificadas o eliminadas.
- Limpieza automática de propiedades que ya no existen en el CRM.

#### 🏠 Custom Post Type Profesional
- Crea automáticamente un Custom Post Type "Property" (Propiedad) optimizado para inmobiliarias.
- Soporte completo para el editor Gutenberg de WordPress.
- Permalinks personalizables para SEO.
- Compatible con cualquier tema de WordPress.

#### 📊 Gestión Completa de Metadatos
El plugin sincroniza y gestiona más de 100 campos diferentes de información de propiedades, organizados en categorías:

- **Características básicas**: Referencia, tipo de operación, tipo de propiedad, ubicación, dirección completa.
- **Superficies**: Metros útiles, construidos, parcela, terraza, cocina, salón, patio, buhardilla, sótano.
- **Distribución**: Habitaciones, baños, aseos, salones, plazas de garaje, plantas, año de construcción.
- **Datos de la propiedad**: Estado de conservación, carpintería, suelo, orientación, vistas, calefacción, agua caliente, electrodomésticos.
- **Certificación energética**: Certificado, letra energética, valor energético, emisiones, referencia del certificado.
- **Características especiales**: Más de 50 características adicionales (ascensor, piscina, jardín, garaje, trastero, etc.).
- **Entorno**: Proximidad a servicios (colegios, hospitales, transporte público, playa, montaña, etc.).
- **Datos comerciales**: Precios, honorarios, IVA, comisiones, exclusivas, valoraciones.

#### 🎯 Sistema de Mapeo de Campos Personalizado
- Interfaz visual para mapear campos del CRM a campos personalizados de WordPress.
- Asignación flexible de variables del CRM a metadatos de WordPress.
- Permite personalizar completamente cómo se almacenan los datos en tu sitio.

#### ⚡ Importación Manual con Interfaz AJAX
- Botón de importación manual desde el panel de administración.
- Proceso de sincronización en tiempo real con barra de progreso.
- Log detallado de todas las operaciones realizadas.
- Permite controlar exactamente cuándo se sincronizan las propiedades.

#### 🔧 Panel de Administración Intuitivo
- Interfaz organizada en pestañas para fácil navegación:
  - **Importar Propiedades**: Control manual de la sincronización.
  - **Configuración**: Ajustes de conexión API, tipo de CRM, post type y slugs.
  - **Fusionar Variables**: Sistema de mapeo de campos personalizado.
- Visualización de todos los metadatos de cada propiedad en metaboxes organizados.

#### 🛡️ Gestión Inteligente de Propiedades
- Detección automática de propiedades duplicadas.
- Actualización de propiedades existentes sin crear duplicados.
- Envío automático a papelera de propiedades que ya no existen en el CRM.
- Mantiene la integridad de los datos durante todo el proceso.

#### 🎨 Compatibilidad Total
- Funciona con cualquier tema de WordPress.
- Compatible con el editor Gutenberg.
- No requiere plugins adicionales.
- Código limpio y optimizado siguiendo los estándares de WordPress.

## Dependencias y Funcionalidades

Este plugin **no requiere plugins adicionales** para funcionar. Es completamente autónomo y utiliza únicamente las funcionalidades nativas de WordPress.

### Requisitos del Sistema
- WordPress 5.0 o superior
- PHP 7.4 o superior
- Acceso a la API de Inmovilla o Anaconda CRM con credenciales válidas

### Integraciones Soportadas
- **Anaconda CRM**: Integración completa mediante API REST con autenticación Bearer Token.
- **Inmovilla CRM**: Integración mediante API web con sistema de autenticación personalizado.
