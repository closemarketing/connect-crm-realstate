# Connect CRM Real State - Commercial Description

## Complete Commercial Description

**Connect CRM Real State** is a professional WordPress plugin designed to connect and synchronize real estate properties from external CRM systems directly to your WordPress website. It's the perfect solution for real estate agencies, property agencies, and developers who need to automate the management of their real estate catalog.

### What does the plugin do?

This plugin acts as an intelligent bridge between your real estate CRM system (Inmovilla or Anaconda) and WordPress, allowing all your properties to be automatically synchronized on your website without the need to manually copy and paste information.

### Who is it aimed at?

- **Real estate agencies and property agencies** that use CRM systems such as Inmovilla or Anaconda and need to display their properties on their WordPress website.
- **Web developers** who work with clients in the real estate sector and need a professional integration solution.
- **Real estate website owners** looking to automate the update of their property catalog.

### What problem does it solve?

The real estate sector requires constantly keeping the property catalog updated on the website. Doing it manually is:
- **Time-consuming**: Copying and pasting information from each property from the CRM to the website.
- **Error-prone**: Outdated information, incorrect data, or duplicate properties.
- **Inefficient**: Not scalable when you have hundreds or thousands of properties.
- **Costly**: Requires dedicated staff just to keep the website updated.

**Connect CRM Real State** solves all these problems by completely automating the synchronization process.

### Key Benefits and Main Features

#### 🔄 Automatic Bidirectional Synchronization
- Connect your WordPress with the most used CRM systems in Spain: **Inmovilla** and **Anaconda**.
- Automatic synchronization of all properties from the CRM API.
- Intelligent update: detects new, modified, or deleted properties.
- Automatic cleanup of properties that no longer exist in the CRM.

#### 🏠 Professional Custom Post Type
- Automatically creates a "Property" Custom Post Type optimized for real estate agencies.
- Full support for WordPress Gutenberg editor.
- Customizable permalinks for SEO.
- Compatible with any WordPress theme.

#### 📊 Complete Metadata Management
The plugin synchronizes and manages more than 100 different property information fields, organized into categories:

- **Basic features**: Reference, operation type, property type, location, complete address.
- **Surfaces**: Useful square meters, built, plot, terrace, kitchen, living room, patio, attic, basement.
- **Distribution**: Bedrooms, bathrooms, toilets, living rooms, parking spaces, floors, construction year.
- **Property data**: Conservation status, carpentry, floor, orientation, views, heating, hot water, appliances.
- **Energy certification**: Certificate, energy rating, energy value, emissions, certificate reference.
- **Special features**: More than 50 additional features (elevator, pool, garden, garage, storage, etc.).
- **Environment**: Proximity to services (schools, hospitals, public transport, beach, mountain, etc.).
- **Commercial data**: Prices, fees, VAT, commissions, exclusives, valuations.

#### 🎯 Custom Field Mapping System
- Visual interface to map CRM fields to WordPress custom fields.
- Flexible assignment of CRM variables to WordPress metadata.
- Allows complete customization of how data is stored on your site.

#### ⚡ Manual Import with AJAX Interface
- Manual import button from the administration panel.
- Real-time synchronization process with progress bar.
- Detailed log of all operations performed.
- Allows you to control exactly when properties are synchronized.

#### 🔧 Intuitive Administration Panel
- Interface organized in tabs for easy navigation:
  - **Import Properties**: Manual control of synchronization.
  - **Settings**: API connection settings, CRM type, post type and slugs.
  - **Merge Variables**: Custom field mapping system.
- Display of all metadata for each property in organized metaboxes.

#### 🛡️ Intelligent Property Management
- Automatic detection of duplicate properties.
- Update of existing properties without creating duplicates.
- Automatic sending to trash of properties that no longer exist in the CRM.
- Maintains data integrity throughout the process.

#### 🎨 Full Compatibility
- Works with any WordPress theme.
- Compatible with Gutenberg editor.
- No additional plugins required.
- Clean and optimized code following WordPress standards.

## Dependencies and Features

This plugin **does not require additional plugins** to function. It is completely autonomous and uses only WordPress native functionalities.

### System Requirements
- WordPress 5.0 or higher
- PHP 7.4 or higher
- Access to Inmovilla or Anaconda CRM API with valid credentials

### Supported Integrations
- **Anaconda CRM**: Complete integration via REST API with Bearer Token authentication.
- **Inmovilla CRM**: Integration via web API with custom authentication system.
